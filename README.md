## 📥 Downloads

You can run Astra Browser in different ways:

### 🪟 Windows

Download the latest **AstraBrowser.exe** from the **Releases** page.

No installation required.

### 🐧 Linux

You can choose one of the following:

- **AstraBrowser.deb** – for Debian/Ubuntu based distributions.
- **AstraBrowser.AppImage** – works on most Linux distributions without installation.

Or run the project from source:

```bash
git clone https://github.com/YOUR_USERNAME/AstraBrowser.git
cd AstraBrowser

python3 -m venv venv
source venv/bin/activate

pip install -r requirements.txt

python main.py
```
